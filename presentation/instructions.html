Method 1: File Upload (Recommended - No Setup Required)

  1. Open the game in your browser
  2. Click the "Upload PDF" button in the top-left corner
  3. Select your PDF file
  4. The game will process and display your slides automatically

Method 2: Folder Method (Requires Local Server)

  • Save your PDF file in this /presentation/ folder with one of these names:
    • presentation.pdf (recommended)
    • slides.pdf
    • slide.pdf
    • deck.pdf
    • talk.pdf
  • Run a local server (see SERVER_SETUP.md for details)

Why Two Methods?

CORS Security Restriction: Modern browsers prevent websites from accessing local files directly for security reasons. This causes "CORS policy" errors when trying to load PDFs from the file system.

Solutions:
  • File Upload: Works everywhere, no setup needed
  • Local Server: Allows folder-based loading, requires server setup

How it works:

  1. Place your PDF in the /presentation/ folder
  2. Start the game
  3. The game will automatically detect and load your PDF
  4. Each PDF page becomes a game room/slide
  5. Navigate between slides using A/D keys
  6. Click "Reload PDF" button to refresh after changing PDFs
Supported PDF names:
./presentation/presentation.pdf
./presentation/slides.pdf
./presentation/slide.pdf
./presentation/deck.pdf
./presentation/talk.pdf

Troubleshooting:

  • CORS Errors: Use the "Upload PDF" button instead of the folder method
  • PDF doesn't load: Check the browser console for errors
  • For encrypted PDFs: You'll be prompted for the password
  • Large PDFs: May take a moment to process
  • No PDF found: The game uses built-in placeholder slides
  • Server setup needed: See SERVER_SETUP.md for local server instructions
